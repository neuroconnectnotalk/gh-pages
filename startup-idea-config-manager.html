STARTUP IDEA - config manager 

sudosadhu

  • these all apps although provide great functionalituy but they are hard to use in the sense that these apps do not provide the ability to sync and manage the customisations and backing and managing up takes a lot of time  
  • the idea is to create a  software that runs in background and recognises all these personal changes in all the different apps and then continuously keep maintaning a single configuration file for the entire apps,. It manages all the updates that happen in the background , and stores the configuration in the background to the external trusted storage
  • so when you open the app you will do these 3 things 
    • allow the app to sync and use permissions 
    • select the storage
    • select the apps the confiuraion file you want to manage (for ex- firefox , freetube , keepass, smart rss, brave  sync chain .... etc, )
